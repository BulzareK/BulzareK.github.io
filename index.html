Acerca de mí

El mundo de los computadores siempre ha estado presente en mi vida. De manera autodidacta aprendí a armar PCs y configurarlos. Luego, leyendo códigos html, pude entender de manera básica las estructuras de la programación. Tengo varias carreras cursadas con éxito. Soy titulado de Traducción e Interpretación en inglés y también poseo título de Pedagogía y Licenciatura en Educación. A pesar de haber estado siempre ligado a las artes y al área humanista, siempre ha habido en mí un gran interés por descubrir el mundo de la programación. Espero que Desafío Latam me dé las herramientas necesarias para poder incursionar en este ámbito laboral

Traductor-Intérprete-Profesor

Estudié la carrera de traducción e interpretación en inglés y me desempeñé en empresas de la zona por un par de años antes de estudiar pedagogía. Actualmente llevo 22 años en la docencia.

Músico-compositor

Desde muy joven estuve ligado a la música. Me formé en orquestas para eventos y participé de diferentes grupos en diversos estilos musicales. Mi especilidad es el folklore latinomamericano y la cueca chilena.

Estudiante de Programación

Ingresé a estudiar programación alentado por amigos programadores y mi familia, quienes vieron en mí ciertas aptitudes que se requieren en el campo de la programación. Espero que mi camino esté lleno de desafíos que pueda ir superando poco a poco.

Trabajos

HTML5

Conocimientos avanzados de HTML5, lenguaje de etiquetas utilizado para la estructuración y la presentación de contenido en los sitios web.

CSS3

El CSS podría definirse como un tipo de lenguaje que permite definir y crear la presentación de un documento ya estructurado y escrito en un lenguaje de marcado como puede ser HTML. Es decir, permite generar el diseño visual de páginas web e interfaces de usuario.

Bootstrap 5

Bootstrap es una biblioteca multiplataforma o conjunto de herramientas de código abierto para diseño de sitios y aplicaciones web.

Git & Github

GitHub es una forja para alojar proyectos utilizando el sistema de control de versiones Git. Se utiliza principalmente para la creación de código fuente de programas de ordenador. El software que opera GitHub fue escrito en Ruby on Rails.

Javascript

JavaScript es un lenguaje de programación interpretado, dialecto del estándar ECMAScript. Se define como orientado a objetos, basado en prototipos, imperativo, débilmente tipado y dinámico.

React JS

React es una biblioteca Javascript de código abierto diseñada para crear interfaces de usuario con el objetivo de facilitar el desarrollo de aplicaciones en una sola página. Es mantenido por Facebook y la comunidad de software libre.
